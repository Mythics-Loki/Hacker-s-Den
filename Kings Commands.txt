### 🧨 Chat Control (messages table)

```
wipe
```

Deletes all messages
(Updates: messages table)

---

### 🔇 Global Mute Control (den_settings table)

```
muteall
unmute
```

Controls `global_mute`

---

### 🎨 Theme Control (den_settings table)

```
bloodmode
greenmode
```

Updates `theme`

---

### 🌀 Animation Control (den_settings table)

```
chaos
calm
```

Updates `chaos_mode`

---

### 💀 Cinematic Control (den_settings table)

```
override
```

Updates `system_override`

---

### 📢 Global Announcement (den_settings table)

```

```

---

### 🚨 System Freeze (den_settings table)

```

```

Updates `lockdown`

---

### 👑 Role Management (profiles table)

```

```

Updates:

* role
* role_level

---

### ⚡ Matrix Speed Control (den_settings table)

